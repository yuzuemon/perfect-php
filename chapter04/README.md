if 構文
```
if (式)
  文
```

```
if (式) {
  文
  文
  ...
}
```

if 文
```
if (条件式)
  文
```

```
if (条件式)
  文1
else
  文2
```

```
if (条件式1)
  文1
elseif (条件式2)
  文2
else
  文N
```

```
while (条件式)
  文
```

```
do
  文
while (条件式);
```

```
for (初期化式; 条件式; 反復式)
  文
```

```
// value を取得
foreach (反復可能なデータ構造 as 要素)
  文


// key, value を同時に取得
foreach (反復可能なデータ構造 as キー => 要素)
  文

// 反復時に参照を用いる
foreach (反復可能なデータ構造 as &要素)
  文
```
